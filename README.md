
# 📡 GSM Based water pump  Control System

## 📌 Overview
This project allows remote control of electrical devices using GSM (missed call and SMS) without internet.

## 🚀 Features
- Missed call based control
- SMS control
- Authorized user security
- No internet required

## ⚙️ Components
- Arduino
- SIM800L GSM Module
- Relay Module
- Power Supply
- water pump

## 🔧 Working
1. User gives missed call or sends SMS  
2. GSM module receives signal  
3. Arduino checks authorized number  
4. Relay turns ON/OFF
